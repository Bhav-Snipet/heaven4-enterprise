## [0.1.0] — Platform Foundation
### Added
- Spring Boot 3.5 backend scaffold (Java 21, Maven, Port 8085)
- React + TypeScript + Vite frontend scaffold
- PostgreSQL + Flyway migration V1 (platform tables)
- All 19 Platform Engine interfaces
- Exception hierarchy (Business, Validation, Security, Resource, Infrastructure)
- Event framework (Base, Domain, Integration, Notification, Audit events)
- Security skeleton (JWT, WorkspaceResolver, PermissionResolver)
- All 6 workspace layouts (Customer, Employee, Kitchen, Manager, Admin, Owner)
- Shared component library foundation
- Design system tokens
- Docker Compose for local PostgreSQL
- Health and Actuator endpoints
- Swagger / OpenAPI documentation
